# **Cinefinity**

Cinefinity es una plataforma web destinada a simular la experiencia completa de compra de entradas de cine. Su objetivo final es permitir a los usuarios explorar películas, seleccionar horarios y asientos, gestionar un carrito de compra y completar transacciones de manera interactiva.

Actualmente, el proyecto se encuentra en fase muy temprana de desarrollo, con la API implementada para la gestión de usuarios y películas.

### **Objetivos Futuros**

---

* Implementar selección de horarios y salas de cine, con disponibilidad en tiempo real.
* Desarrollar la selección de asientos mediante interfaz visual interactiva.
* Integrar un carrito de compra y flujo de transacción, simulando procesos reales de venta de entradas.
* Optimizar la experiencia de usuario en dispositivos móviles y escritorio.
